## Quick Install
#### Warning! Linux only
> git clone git@github.com:AivazianArtur/puring.git \
> cd puring \
> make install


#### In v0.2 you could install via pip, for now you should import library like this:

``` 
import sys

sys.path.insert(0, 'build/lib.linux-x86_64-cpython-312')

import puring
```
